Chapter2 Linux 基础命令
===============================
## Section 2.1 Linux 目录结构
### 1.Linux目录结构
- Linux 的目录结构是一个树形结构；
- Windows拥有多个盘符，如C盘，D盘等。但是Linux没有这个概念，只有一个根目录`\`，所有文件都在它下面。
### 2.Linux路径的描述方式
- 在Linux系统中，路径之间的层级关系使用`/`表示；出现在开头的`/`表示根目录，后面的`/`表示层级关系；
- 在Windows系统中，路径之间的层级关系使用`\`表示；

**例子** 在根目录下有一个文件夹test，文件夹内有一个文件叫hello.txt，请描述文件路径：
```linux
/test/hello.txt
```

## Section 2.2 Linux命令入门
### 1.Linux 命令的基础
- 命令行：即Linux终端，是一种命令提示符页面。以“纯字符”的形式操作系统，可以使用各种字符化命令对系统发出操作指令；
- 命令：即Linux程序，一个命令就是一个Linux程序。命令没有图形化页面，可以在命令行提供字符化的反馈。
- 命令基础格式：
  ```linux
    command [-options] [parameter]
  ```
  - `command`：命令本身
  - `-options`：[可选，非必须] 命令的一些选项，可以通过选项控制命令的行为细节；
  - `parameter`：[可选，非必须] 命令的参数，多数用于命令的指向目标等；
**例子**
```linux
ls -l /home/test
```
其中`ls`是命令，`-l`是选项，这一句命令表示以列表的形式显示 `/home/test`目录中的内容；

### 2.ls(list files)命令
#### (1) `ls`命令语法
`ls`命令的作用是列出目录下的内容，语法细节如下：
```
ls [-a -l -h] [linux 路径]
```
- **基本命令**： 当不使用选项和参数，直接使用ls本身，则以平铺的形式列出当前工作目录下的内容；
- **选项**：
  | 选项  | 作用 |
  | :---: | :---:|
  |`-a`|可以查看所有文件（包括隐藏文件，以`.文件名`表示）；|
  |`-l`|以列表竖向排列的形式展示内容，并展示更多信息；|
  |`-h`|以易于阅读的方式列出文件的大小|
- **参数**： 当使用参数时，ls命令将会列出指定工作目录下的内容；比如可以使用下面的命令来查看根目录的内容
```
ls /
```
- **注意**: 选项可以混合使用，下面三种用法都是等价的
  ```
  ls -l -a
  ```
  ```
  ls -la
  ```
  ```
  ls -al
  ```

#### (2) home目录
- Linux系统的命令行终端在执行命令时，需要一个**当前工作目录**；
- 在启动的时候会默认加载home目录且将其作为当前工作目录，所以在上面的命令后列出的时home目录的内容；
- **home目录**：每个Linux操作用户在Linux系统的个人账户目录路径在：`/home/用户名`；
- 在Windows和Linux系统下都设有home目录；

### 3.cd(change directory)/pwd(print work diewctory)命令
#### (1) cd命令语法
```
cd [linux路径]
```

- cd命令无需选项，只有参数，表示要切换到哪个目录下；
- cd命令直接执行，不写参数，表示回到用户的Home目录；

#### (2) pwd命令语法
```
pwd
```
注意：pwd命令无选项，无参数。

### 4.相对路径，绝对路径和特殊路径符
#### (1) 相对路径与绝对路径
- **绝对路径**以根目录为起点，路径描述以`/`开头；
- **相对路径**以当前目录为起点，路径描述无需以`/`开头；

**例子**:比如我们想要进入位于home目录下的Desktop文件夹
绝对路径写法
```
cd /home/用户名/Desktop
```
相对路径写法
```
cd
cd Desktop
```

#### (2) 特殊路径符
|特殊路径符|意义|例子|
|:---:|:---:|:---:|
|`.`|表示当前目录|`cd ./Desktop` 表示切换到当前目录下的`Desktop`目录内|
|`..`|表示上一级目录|`cd ..`表示切换到上一级目录，`cd ../..`表示切换到上两级目录|
|`~`|表示Home目录|`cd ~`表示切换到Home目录，`cd ~/Desktop`和`cd /home/用户名/Desktop`一样|

### 5.mkdir(make directory)命令
通过`mkdir`命令可以创建新的目录。
```
mkdir [-p] linux路径
```
- 参数必填，表示linux路径，即要创建的文件夹的路径，相对路径和绝对路径均可；
- `-p`选项可选， 表示自动创建不存在的父目录，适用于创建连续多层级的目录；

**注意**：创建文件夹需要修改权限，请确保操作均在home目录内，不要在home目录外操作，否则无法成功创建，这涉及到权限问题。
```
mkdir /test
```
会返回**权限不够**提示。

### 6.文件操作命令
#### （1）touch命令
可以通过touch 创建文件
```
touch linux路径
```
- 无选项，参数必须填写，表示要创建的文件路径，相对路径，绝对路径等均可；

#### (2) cat/ more命令
有了文件之后，可以通过cat命令查看文件的内容；
```
cat linux路径
```
- cat 没有选项，只有必填参数；

`more`命令和`cat`一样可以查看文件内容且语法一样，但是有如下区别
- `cat` 会将所有文件全部显示出来；
- `more`支持分页查看，按**空格**可以翻页，按**q**可以退出翻页查看；

#### (3) cp命令
cp命令可以用于复制文件\文件夹，cp命令来自于单词copy
```
cp [-r] 参数1 参数2
```
- 选项`-r`可选，用于复制文件夹使用，表示递归
- 参数1， linux路径，表示被复制的文件
- 参数2， linux路径，表示要复制去的地方

#### (4) mv命令
mv命令可以用于移动文件\文件夹，mv命名来源于单词move
```
mv 参数1 参数2
```
- 参数1，linux路径，表示被移动的文件地址；
- 参数2，linux路径，表示要移动去的地方；

注意若移动的目录不存在，则默认成改名效果。

#### (5) rm命令
rm命令用于删除文件\文件夹，rm命名来源于单词remove
```
rm [-r -f] 参数1 参数2 ...... 参数N
```
**命令选项**
|选项|作用|
|:---:|:---:|
|-r|同cp命令一样，可以用于删除文件夹|
|-f|表示force，强制删除，不会弹出提示确认信息。普通用户删除文件不会跳出提示，只有root用户删除文件时会出现提示，所以一般普通用户用不上。|

通过以下命令可以临时切换为/退出root用户：
```
su - root
```
```
exit
```

**模糊匹配**
rm命令支持通配符`*`，其可以匹配任意内容（包括空格）。
|示例|效果|
|:---:|:---:|
|`test*`|表示匹配任何以test开头的内容|
|`*test`|表示匹配任何以test结尾的内容|
|`*test*`|表示匹配任何包含test的内容|

**警告**：`rm`是一个危险的命令，特别是在处于root用户的时候。请千万不要在root管理员用户下执行如下语句：
```
rm -rf/
rm -rf/*
```
其表示从根目录上强制删除目录（删库跑路），效果等同于在windows上执行C盘格式化。

### 7.查找命令
#### (1)which 命令
在上面的学习的Linux命令，其实本体就是一个个的二进制可执行程序，和Windows系统中的`.exe`文件。我么可以通过`which`命令，查看所使用的一系列命令的程序文件存放在哪里。
```
which 要查找的命令
```
#### (2)find命令
我们可以通过find命令去搜索指定的文件。

**方式1： 以文件名的模式开始查找**
```
find 起始路径 -name "被查找文件名"
```
- 为了拥有最大权限，可以在整个系统完成搜索，我们可以切换到root用户以获得管理员权限；
- `find`命令也可以使用通配符`*`;

**方式2： 以文件大小的模式开始查找**
```
find 起始路径 -size +|-n[kMG]
```
- +,-表示大于/小于
- n表示大小数字
- kMG表示大小单位，k表示kb，M表示Mb，G表示Gb

**例子**
- 查找小于10kb的文件： `find / -size -10k`
- 查找大于100Mb的文件：`find / -size +100M`
- 查找大于1Gb的文件：`find / -size +1G`


### 8.管道符
#### (1)grep命令
可以通过grep命令，从文件中通过关键字过滤文件行。
```
greo [-n] 关键字 文件路径
```
- 选项`-n`可选，表示在结果中显示匹配的行号；
- 参数，关键字，必填，表示过滤的关键字；
- 参数，文件路径，必填，表示要过滤的文件，也可以使用管道符来输入；

#### (2)wc命令
可以通过wc命令统计文件的行数，单词数量等
```
wc [-c -m -l -w] 文件路径
```

**命令选项**
|选项|功能|
|:---:|:---:|
|`-c`|统计bytes数量|
|`-m`|统计字符数量|
|`-l`|统计行数|
|`-w`|统计单词数|

**参数**：文件路径，被统计的文件，也可以使用管道符来输入；

#### (3)管道符`|`
管道符可以将左边命令的结果，作为右边命令的输入。

### 9.`echo`，`tail`以及重定向符
#### (1)`echo`命令
`echo`命令可以在命令行内输出指定内容
```
echo 输出内容
```
- 无需选项，只有一个参数表述要输出的内容

#### (2)反引号```
我们可以通过将命令用反引号输出，被```包围的内容，将会被当作命令输出；

#### (3)重定向符
- `>`：将左侧命令的结果，**覆盖**写到符号右侧指定的文件中；
- `>>`：将左侧命令的结果，**追加**写到符号右侧指定的文件中；

#### (4)`tail`命令
利用tail命令，可以查看文件尾部内容，跟踪文件的最新更改
```
tail [-f -num] linux路径
```
- 参数：linux路径，表示被跟踪的文件路径；
- 选项：-f表示持续跟踪；
- 选项：-num表示查看尾部多少行，不写表示10行；
